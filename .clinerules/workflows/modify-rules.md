# Modifying League Rules/Points

To change the point system or format text:

1.  **Edit Rules:** Open `src/data/leagueData.js`.
2.  **Update Points:** Modify the `rules.points` object.
    *   `championship`, `court2`, etc. keys map to the rank (1, 2, 3, 4) on that court.
3.  **Update Text:** Modify `rules.format` or `rules.general` for text changes on the Format page.
